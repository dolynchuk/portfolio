lun.ua search website

The largest Ukrainian Prop-Tech platform featuring interactive 3D maps.

The website underwent a complete overhaul, transitioning from a Jquery + PHP architecture to a modern React SSR application with a Python API. The maps engine was migrated from Google Maps to MapBox, while the website design and user experience were fully revamped. With the addition of new features and an improved UX, the platform attracted more users and significantly increased ad revenue.

Bird rent realtime price chart

A real-time platform displaying rental price statistics for apartments in major Ukrainian cities.

Originally created as a promotional page for the Bird Rent app, the site provided detailed rental price data for 1, 2, and 3-room apartments. Due to its growing popularity, it has since evolved into an independent service that users now rely on to track and compare rental prices.
